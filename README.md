# Product Comparison App

Responsive smartphone comparison tool. React + Material UI.

## Screenshots
![Desktop](screenshots/desktop.png)
![Mobile](screenshots/mobile.png)
![Comparison](screenshots/compare.png)

## Features
- Grid of 8 phones (image, price, specs)
- Select up to 3 for side-by-side table
- Highlights best specs (green)
- Search, dark mode, keyboard access
- localStorage saves selections

## Quick Setup
```bash
npm install
npm start
```
